데이터타입의 종류

기본 타입(Primitive Type) -불변성을 뛴다. 불변성이란 - 변하지 아니하는 성질.
Number, String, boolean, null, undefined, Symbol

기본형은 할당이나 연산시 복제된다.
값이 담긴 주솟값을 바로 복제

참조 타입(Reference Type)
object, array, function, date, regExp, map, set

연산시 참조 된다.
참조형은 값이 담긴 주솟값들로 이루어진 묶음을 가리키는 주솟값을 복제 -(가리키는 주솟값)


데이터 타입에 관한 배경지식

메모리와 데이터
컴퓨터는 모든 데이터를 0 또는 1로 바꿔서 기억합니다. 0또는 1만 표현할 수 있는 하나의 메모리 조각을 비트라고한다.
메모리는 비트들로 구성되어있다.
각 비티는 고유한 식별자는 지닌다.
각 비트는 고유한 식별자를 통하여 위치를 확인할 수 있다. - 비트단위로 확인하는 것은 비효율적 그래서 byte라는 단위가 생김
1바이트는 8개의 비트로 구성되어있다.

식별자와 변수
변수 variable
식별자 identifier
변수와 식별자를 혼용하는 경우가 많습니다. -혼용이 가능한 이유 대부분의 경우에 문맥에 따라 무엇을 말하고자 하는지를 유추할 수 있기 때문

변수 선언과 데이터 할당
변수란 
변경 겨능한 데이터가 담길 수 있는 공간 또는 그릇.

변수 선언
var a;  - 이 데이터의 식별자는 a로한다. 선언할 때는 undifined /
이 이공간에 숫자를 담았다가 문자열을 담는 등의 다양한 명령을 내릴 수 있다.

데이터 할당
var a; // 변수 a선언
a = 'abc' // 변수 a에 데이터 할당
var a = 'abc'// 한문장
a라는 이름을 가진 주소를 검색해서 그곳에 문자열 abc를 할당
실제로는 직접 저장하지 않습니다.  주소를 변수영역에 저장하는 식
